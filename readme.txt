== Description ==

Adds support for the H language in SyntaxHighlighter Evolved plugin. As used in mobile app ["Hacked"](http://www.hackedapp.com).

Plugin includes a Hackpad  theme to make your code samples resemble what they would look like in the Hacked app Hackpad UI.

**Requires SyntaxHighlighter Evolved plugin to be installed first.**

You can see it in action on this blog post: [Hacked Android App Coding Solutions](http://andrewjohns.net/blog/2015/08/hacked-app-android-coding-solutions)

== Installation ==

1. Upload shBrushHackpad to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. **(Optional)** Enable the *Hacked* theme in the SyntaxHighlighter plugin

== Frequently Asked Questions ==

= What aliases can I use? =

*hack*, *hacked*, or *hackpad* will all work.

= How do I use this brush in my code block? =

`[code lang="hack"]
// some H code will go in here.
[/code]`
